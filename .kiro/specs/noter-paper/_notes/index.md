- 图编号双向闭合（design.md Property 1）：每张图必须在 `paper/noterPaper.md` 出现 `图 X.Y 图名` 标签且至少被「如图 X.Y 所示」引用一次，并镜像到 `paper/noterChart.md`。
- 表编号一致：每张表必须在正文出现 `表 X.Y 表名` 标签且至少被「如表 X.Y 所示」引用一次。
- 参考文献双向闭合（design.md Property 2）：每条 `[N]` 在正文与文末双向对账，按首次出现顺序连续递增、不跳号、不重号。

每完成一张图 / 表 / 引用的写入，请同步更新本文件对应行的「状态」「是否已镜像到 noterChart.md」「文末是否登记」字段。
